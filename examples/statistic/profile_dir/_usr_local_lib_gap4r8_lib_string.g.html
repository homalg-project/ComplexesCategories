50##  <#GAPDoc Label="IsString">
51##  <ManSection>
52##  <Filt Name="IsString" Arg='obj'/>
53##
54##  <Description>
55##  A <E>string</E> is a dense list (see&nbsp;<Ref Func="IsList"/>,
56##  <Ref Func="IsDenseList"/>) of characters (see&nbsp;<Ref Func="IsChar"/>);
57##  thus strings are always homogeneous
58##  (see&nbsp;<Ref Func="IsHomogeneousList"/>).
59##  <P/>
60##  A string literal can either be entered as the list of characters
61##  or by writing the characters between <E>doublequotes</E> <C>"</C>.
62##  &GAP; will always output strings in the latter format.
63##  However, the input via the double quote syntax enables &GAP; to store
64##  the string in an efficient compact internal representation.
65##  See <Ref Func="IsStringRep"/> below for more details.
66##  <P/>
67##  Each character, in particular those which cannot be typed directly from
68##  the keyboard, can also be typed in three digit octal notation.
69##  And for some special characters (like the newline character) there is a
70##  further possibility to type them,
71##  see section <Ref Sect="Special Characters"/>.
72##  <P/>
73##  <Example><![CDATA[
74##  gap> s1 := ['H','e','l','l','o',' ','w','o','r','l','d','.'];
75##  "Hello world."
76##  gap> IsString( s1 );
77##  true
78##  gap> s2 := "Hello world.";
79##  "Hello world."
80##  gap> s1 = s2;
81##  true
82##  gap> s3 := "";  # the empty string
83##  ""
84##  gap> s3 = [];
85##  true
86##  gap> IsString( [] );
87##  true
88##  gap> IsString( "123" );  IsString( 123 );
89##  true
90##  false
91##  gap> IsString( [ '1', '2', '3' ] );
92##  true
93##  gap> IsString( [ '1', '2', , '4' ] );  # strings must be dense
94##  false
95##  gap> IsString( [ '1', '2', 3 ] );  # strings must only contain characters
96##  false
97##  ]]></Example>
98##  </Description>
99##  </ManSection>
